## Comprendre la Mécanique de Google

Avant les outils, comprendre pourquoi une stratégie à 90% est possible.

Google ne classe pas des pages. Il classe des **sites qui font autorité sur un sujet**. Si Win Agro devient la ressource la plus complète sur l'agriculture béninoise, Google n'a pas le choix — il l'affiche en premier pour toutes les requêtes du domaine.

C'est ce qu'on appelle la **Topical Authority**. Trois conditions pour l'obtenir :

1. **Couverture** — répondre à TOUTES les questions que quelqu'un pourrait poser sur le sujet
2. **Profondeur** — chaque réponse doit être meilleure que ce qui existe déjà
3. **Structure** — Google doit comprendre que tout le contenu forme un seul écosystème cohérent

Ce document est la carte de cet écosystème.

## 03 — SEO On-Page — Règles de Production

### Template de page pilier (appliqué à chaque silo)

```markdown
# [Mot-clé principal H1] — Win Agro Agri Tech Solutions

Meta title    : [Mot-clé] au Bénin — Formation & Accompagnement | Win Agro
Meta desc     : [Promesse en 155 caractères max incluant mot-clé + CTA]
Canonical     : https://winagro.bj/[slug]/
Schema.org    : Service + LocalBusiness + FAQPage

## [H2 — Mot-clé secondaire]
[300 mots minimum — répondre à l'intention primaire]

## [H2 — Sous-thème 1]
[200 mots]

## [H2 — Sous-thème 2]
[200 mots]

## Questions fréquentes (FAQ — Featured Snippets)
### [Question 1 contenant le mot-clé]
### [Question 2]
### [Question 3]

## [CTA de conversion]
```

### Règles absolues pour chaque page

- **Mot-clé principal** dans le H1, les 100 premiers mots, l'URL, la meta title, la meta description
- **Densité** : mot-clé principal 1 à 2% du texte — jamais plus
- **Variations sémantiques** : 5 à 8 synonymes et expressions connexes par page
- **Longueur minimum** : 800 mots pour les pages silo, 1 500 mots pour les articles de blog
- **Images** : 1 image minimum, alt text = variante du mot-clé principal
- **Liens internes** : 3 minimum par page (1 vers parent, 2 vers pages sœurs)
- **CTA** : 1 par section, ancre vers `#contact` ou page de conversion

```javascript
// lib/analytics.ts

export const GA_EVENTS = {
  // Conversion principale
  LEAD_SUBMITTED: 'lead_submitted',

  // Micro-conversions
  WHATSAPP_CLICK: 'whatsapp_click',
  PHONE_CLICK: 'phone_click',
  SERVICE_CTA_CLICK: 'service_cta_click',
  PRODUCT_VIEW: 'product_view',
  BLOG_ARTICLE_READ: 'blog_article_read',  // scroll > 80%
  FAQ_EXPANDED: 'faq_expanded',
  FORMATION_PAGE_VIEW: 'formation_page_view',

  // Engagement
  VIDEO_PLAY: 'video_play',
  TESTIMONIAL_VIEW: 'testimonial_view',
  MAP_INTERACTION: 'map_interaction',
}

// Exemple d'implémentation
export function trackLeadSubmitted(serviceType: string) {
  if (typeof window !== 'undefined' && window.gtag) {
    window.gtag('event', GA_EVENTS.LEAD_SUBMITTED, {
      service_type: serviceType,
      currency: 'XOF',
    })
  }
}

export function trackWhatsAppClick(source: string) {
  if (typeof window !== 'undefined' && window.gtag) {
    window.gtag('event', GA_EVENTS.WHATSAPP_CLICK, {
      click_source: source,  // 'fab', 'hero', 'contact', 'product'
    })
  }
}
```
